🚀 LeetCode Matrix Website

A web application that helps users visualize and analyze their LeetCode performance in a structured matrix format. It fetches user data and displays problem-solving statistics in an interactive and easy-to-understand way.

📌 Features
🔍 Search any LeetCode username
📊 Visual representation of solved problems
🧠 Difficulty-wise breakdown (Easy / Medium / Hard)
⚡ Fast and responsive UI
🌐 API-based data fetching

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript / React
API: LeetCode GraphQL API
Styling: CSS / Tailwind (if used)

🚀 How to Run Locally

Clone the repository

git clone https://github.com/your-username/your-repo-name.git

Navigate to project folder

cd your-repo-name

Install dependencies

npm install

Start the project

npm run dev
